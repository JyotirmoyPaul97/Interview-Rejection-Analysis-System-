A powerful gamified AI platform that analyzes interview rejections, reveals hidden patterns, and transforms your career journey into an engaging, rewarding experience.

---

## ✨ Game-Like UI/UX & Animations

IRAS is designed like a premium video game to make career growth addictive and visually stunning.

### Core Design Philosophy
- **Cyberpunk-Neon + Dark Fantasy** theme with deep blacks, electric purples, vibrant cyans, and glowing accents
- **Immersive HUD-style interface** inspired by RPG games
- **Premium feel** with buttery-smooth 60 FPS animations throughout
- Zero-clutter, high-focus design with micro-interactions on every element

### Epic Animations

- **Hero Landing Page**: 
  - Glowing neon title with subtle glitch and pulse effects
  - Floating 3D icons (resume, microphone, trophy) that orbit smoothly with parallax
  - Particle system where rejection sparks transform into golden XP orbs
  - Large pulsing "Enter the Arena" button with hover ripple and glow effects

- **Dashboard Experience**:
  - Dynamic XP progress bar that fills with energy glow
  - Stat cards that bounce and scale in with satisfying physics
  - Live rejection heat-map with spinning globe and glowing intensity particles
  - Level badge with continuous subtle shine animation

- **Rejection Analysis Report**:
  - Cinematic chest-opening reveal sequence with golden light burst and slow-motion data card fly-ins
  - Interactive 3D Skill Tree with glowing energy beams connecting nodes and auto-camera movement
  - Animated charts that rise like combo meters with staggered timing
  - "Boss Defeated" screen with shockwave effect and particle explosions

- **Level Up & Celebration**:
  - Full-screen "LEVEL UP!" sequence with shockwave, badge spin, and floating XP numbers
  - Physics-based confetti and reward particle explosions on achievements
  - Smooth liquid-metal style progress rings

- **Micro-interactions**:
  - All buttons have press-down + neon trail effects
  - Hover states lift cards with holographic glow
  - Quest cards have smooth drag & drop with snap animation
  - Loading states feature cinematic portal spinning with progress glow

All animations are built with **Framer Motion + GSAP + React Three Fiber**, highly optimized, and fully respect `prefers-reduced-motion` for accessibility.

## 🛠 Tech Stack

- **Frontend**: Next.js 15 + React 19 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui + custom neon utilities
- **Animations**: Framer Motion, GSAP, React Three Fiber, tsparticles
- **AI Engine**: GPT-4o + Claude 3.5
- **Database**: Supabase (PostgreSQL + Realtime)
- **Deployment**: Vercel
